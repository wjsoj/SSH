# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [Unreleased]

### Added
- Initial project setup with Next.js 16, TypeScript, and shadcn/ui
- GitHub OAuth authentication
- Prisma ORM with SQLite database
- Biome code linting and formatting
- Docker and Docker Compose support
- GitHub Actions CI/CD pipelines
- Husky pre-commit hooks
- Vitest testing framework
- Dependabot for dependency updates
- CodeQL security scanning

### Features
- Browse and search skills from GitHub repositories
- View individual skill details with README
- Admin dashboard for repository management
- Automated sync from GitHub repositories
- Repository-level skill listing

### Technical
- React Compiler enabled
- Standalone Docker output
- TypeScript strict mode
- ESLint replaced with Biome

## [0.1.0] - 2026-02-19

### Added
- Initial release
- Basic skill browsing
- GitHub authentication
- Admin dashboard
